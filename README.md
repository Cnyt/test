# BolBol Hybrid - iOS

## Requirements
- Cordova v3.6.3-0.2.13 - https://cordova.apache.org
```
npm install -g cordova@5.2.0
```
- GulpJS v3.9.0 - http://gulpjs.com
```
npm install gulp-cli -g
npm install gulp -D
touch gulpfile.js
```

- TypeScript - https://www.typescriptlang.org
```
npm install -g typescript
```
